# LPM
The package manager for LiteLoaderBDS, made by Lua.

## Features
 - [x] Multi download protocol.
 - [x] Multi repoistroy design.
 - [x] Package storage over Sqlite3.
 - [ ] Install/Remove/Update/Purge software.
 - [ ] Verify packages over GPG.
